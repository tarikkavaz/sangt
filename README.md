# Strapi, Apollo, Nuxt.js, GraphQL, Tailwind CSS | Boilerplate

- Strapi v4.2.3
- Apollo v2.34.0
- Nuxt v2.15.8
- GraphQl v15.8.0
- Tailwind CSS v3.1.6

## Setup:

```bash
git clone https://github.com/tarikkavaz/sangt.git
cd sangt
yarn setup
```

---

## Develop:

```bash
yarn dev
```

open: http://localhost:3000/ & http://localhost:1337/admin/
